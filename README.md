# 📚 SkillSphere

## 🚀 Project Purpose
SkillSphere is an online learning platform where users can explore courses, view trending content, and manage authentication (login/register). The goal of this project is to demonstrate a modern Next.js App Router application with clean UI, dynamic data handling, and user interaction features.

## ✨ Key Features

- 🔐 User Authentication (Sign Up & Login)
- 🔎 Search functionality for courses
- 🔥 Trending Courses section
- 📦 Data fetching from JSON
- ⚡ Fast UI with Next.js App Router
- 🔔 Toast notifications for user actions
- ⏳ Loader while data fetching
- 🚫 Custom 404 Not Found page
- 📱 Responsive design (Mobile + Desktop)

## 🧱 Tech Stack
